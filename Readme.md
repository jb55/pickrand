
# pickrand

  Pick a random element from an array

## Installation

### Component

    $ component install jb55/pickrand

### Node

    $ npm install pickrand

## API

```javascript
var pickrand = require('pickrand') 
var x = pickrand(["bill", "john", "jake"]);

// x can be "bill", "john", or "jake"

var x = pickrand([])

// x is null
```
